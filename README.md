# 📄 Resume Parser & Job Role Predictor

## 🚀 Overview
This project uses NLP and Machine Learning to:
- Extract information from resumes (email, phone, skills)
- Predict job role using TF-IDF and classification model

## 🛠️ Tech Stack
- Python
- NLP (Regex, NLTK)
- Scikit-learn
- Streamlit

## 📂 Features
- Resume parsing (PDF, DOCX, TXT)
- Skill extraction
- Job role prediction
- Confidence score

## ▶️ Run Project

```bash
pip install -r requirements.txt
python -m streamlit run app.py
